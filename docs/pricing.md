# Pricing Direction

## Monthly Plans

- Starter
  - small team / limited environments
  - discovery, risk scan, export
- Growth
  - core migration plan
  - deploy, validate, Git backup, reports
- Agency
  - multi-client workflows
  - higher limits and priority support
- Enterprise
  - self-hosted / private deployment
  - custom security and support

## Value Positioning

- faster migrations than manual AWS-native coordination
- reduced senior engineer time
- audit trail, validation, and reusable migration workflow
